# 量化金融数据智能体

专为量化交易设计的全市场金融数据获取智能体。

## 功能特性

### 1. 多源数据获取 (成功率99%+)
- **腾讯财经** - 实时行情，速度最快
- **新浪财经** - 备用源，稳定可靠
- **雪球** - 个股数据补充

### 2. 实时数据 (毫秒级)
- A股/港股实时行情
- 指数实时数据
- 资金流向

### 3. 历史数据
- 全市场K线数据
- 自定义时间范围

### 4. 特色数据
- 资金流向
- 龙虎榜数据
- 融资融券

## API使用

### 获取指数行情
```
GET /index
```

### 获取个股行情
```
GET /stock/{symbol}
```

### 获取统计数据
```
GET /stats
```

## 数据源优先级

1. 腾讯财经 - 首选，实时最快
2. 新浪财经 - 备用，稳定
3. 雪球 - 辅助补充
